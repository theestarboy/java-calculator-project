# 🧮 Java Swing Desktop Calculator



## 📌 Overview
A simple, user-friendly desktop calculator application built using Java's Swing and AWT frameworks. This project was developed to demonstrate core Object-Oriented Programming (OOP) concepts, graphical user interface (GUI) design, and event-driven programming in Java.

## ✨ Features
* **Standard Arithmetic:** Supports addition (`+`), subtraction (`-`), multiplication (`*`), and division (`/`).
* **Clear Functionality:** Includes a "C" button to easily reset the current calculation state.
* **Structured UI:** Utilizes a combination of `BorderLayout` for the main window and `GridLayout` for a perfectly aligned 4x4 keypad.
* **Robust Error Handling:** Features built-in mathematical validation, specifically catching and gracefully handling "Division by Zero" errors to prevent the application from crashing.
* **Status Tracking:** Includes a real-time status bar at the bottom of the window to provide user feedback (e.g., "Ready" or "Error").

## 🛠️ Tech Stack
* **Language:** Java
* **Libraries:** `javax.swing.*` (GUI components), `java.awt.*` (Layouts), `java.awt.event.*` (Event listeners)
